# 📊 Decathlon Sales Analytics Dashboard  
Power BI | Power Query | DAX  

## 🔹 Project Overview  
Developed a Sales Analytics Dashboard for Decathlon using Power BI to provide clear insights into sales performance across products, categories, countries, and time periods.  

## 🔹 Data Preparation & Modeling  
- Cleaned and transformed raw sales data using Power Query  
- Standardized formats and handled missing values  
- Built a structured data model with DAX measures (Sales, Cost, Profit, Quantity)  

## 🔹 Key Insights  
- Total Sales: $22.4M | Profit: $2.3M | Quantity: 70K  
- Accessories have the highest profit margins  
- Bikes generate the highest revenue with lower margins  
- Q2 is the strongest quarter, Q3–Q4 show seasonal decline  
- The U.S. leads overall sales performance  

## 🔹 Business Recommendations  
- Focus on high-margin categories like Accessories  
- Optimize Bike pricing strategy  
- Run campaigns during low-performing quarters  
- Apply successful U.S. strategies to other regions  

## 🔹 Tools Used  
Power BI | Power Query | DAX | Excel  

## 🔹 Outcome  
An interactive dashboard that supports performance monitoring and data-driven decision-making.
